## Этап 3: Март 2026 — Юридическое оформление и начало разработки

**Статус: ВЫПОЛНЕН (разработка); юр. оформление ожидает**

Разработка backend и frontend для фаз 0–7 завершена полностью и с опережением графика. Реализовано:

- **Фундамент:** 22 миграции, 14 моделей, 14 фабрик, 8 enum-классов, 6 сервисных контрактов, 2 layout (app/guest), 7 UI-компонентов
- **Аутентификация:** SMS-авторизация без пароля (Actions, LogSmsService, Livewire-экраны ввода телефона и кода, rate limiting)
- **Операции:** Учёт доходов/расходов, 16 системных категорий, повторяющиеся операции, API v1, фильтрация по периодам и типам
- **Финансовые цели:** Пошаговая форма создания, детальная страница с метриками, 3 сценария (оптимистичный/базовый/пессимистичный), слайдер «Что если»
- **Инфляция:** Импорт данных ИПЦ Росстата (2023–2025), InflationService (6 методов), расчёт реальной покупательной способности, персональная инфляция по структуре расходов
- **Дашборд:** Четыре итерации (v1: баланс и метрики, v2: лента целей, v3: инфляция и реальный баланс, v4: карточка «Совет дня»)

Юридическое оформление (регистрация ИП, подключение ЮKassa) и настройка боевой инфраструктуры (домен, VPS, SSL) в ожидании.

## Этап 4: Апрель 2026 — Завершение MVP

**Статус: ВЫПОЛНЕН (опережение графика)**

Все основные функциональные блоки MVP реализованы. Помимо ранее выполненных экранов (регистрация, дашборд, операции, цели), завершены:

- **Аналитика и графики (Фаза 5):** AnalyticsService с 4 методами агрегации, API v1 (5 эндпоинтов), компоненты DonutChart и LineChart (ApexCharts), Livewire-страница аналитики с тремя вкладками (расходы по категориям, динамика баланса, персональная инфляция), переключатель периодов, тесты (Pest)
- **AI-советы (Фаза 6):** Rule Engine (5 правил финансового анализа), LlmService (заглушка), AiAdviceService, Artisan-команда `advice:generate`, AiAdviceController + API, frontend (список советов, детали, карточка), дашборд v4 с карточкой «Совет дня», тесты (Pest)
- **Freemium и подписки (Фаза 7):** SubscriptionService, CheckSubscription middleware, FakePaymentService, лимиты Free-плана (50 операций/месяц, 1 цель), экран подписки, PremiumLock, upsell-баннеры, тесты (Pest)
- **Техдолг:** синхронизация InflationServiceInterface с реализацией, переход всех потребителей на интерфейс вместо конкретного класса

**Текущая работа:**

- Профиль и настройки (Фаза 8, ветка `task/ENQ-93`)
